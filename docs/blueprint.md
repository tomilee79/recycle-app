# **App Name**: EcoTrack UI Demo

## Core Features:

- Real-Time Dispatch Dashboard: Display active recycling collection routes, vehicle locations, and task statuses using Firestore mock data.
- Geospatial Monitoring: Visualize collection vehicles on a map interface using Firestore mock data to simulate real-time tracking.
- Settlement Report Visualization: Generate visual reports of recycling collection data (volume, type, location) for settlement purposes, sourcing data from Firestore mock data.
- Material Type Prediction Tool: Use a generative AI tool to predict the type of recycled material being collected based on location and time, using a mock dataset. Intended as a tool to provide helpful suggestions or validation.
- Performance Summary: Provide metrics, generated with mock data from Firestore, on how well resources are used, what materials are collected and when, where problem spots arise.

## Style Guidelines:

- Primary color: Forest green (#0670d4) to represent nature and sustainability.
- Background color: Light beige (#F4F7F9), offering a neutral and clean backdrop.
- Accent color: Sky blue (#87CEEB) for interactive elements and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality; suitable for headlines or body text
- Use simple, line-based icons representing different types of recyclable materials and vehicles.
- Dashboard layout should be clean and intuitive, with clear visual hierarchy and prioritization of key information.
- Subtle animations for data updates and map interactions to provide feedback without being distracting.
